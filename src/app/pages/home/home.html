What do we do?

Deployment as a Service is a solution for developers and enterprises which helps them leverage various cloud platforms for deployment of the application without worrying about deployment process. Our platform will help the enterprise to move from one cloud to another or private cloud to public cloud. DaaS deploys a micro-service application with Kubernetes as container orchestration and Docker as container engine. On top of service, we also storage management layer so that a user could plug-and-play any existing portable storage.

How we do?

Daas is a platform where the user can provide existing kubernetes cluster details to read cluster information. Next, select the destination cloud configuration and create Kubernetes cluster. Upon successful cluster creation, the application is migrated based on information read from existing cluster. Once deployed, the user can manage and monitor the cluster and even scale up/ scale down or remove the deployment.
